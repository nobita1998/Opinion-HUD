在 X 上一眼找到可交易的市场

Opinion HUD 是一个 Chrome 扩展：当推文提到某个项目/事件时，在推文头部显示一个入口，打开面板即可看到匹配的市场与各选项， 并一键打开交易页。
  • 入口位置：在推文头部 “…” 按钮左侧
  • 使用方式：点击入口打开/关闭面板（也支持悬停打开）
  • 面板内容：展示匹配市场与选项，并提供 “Trade” 按钮
查看 GitHub 反馈问题（Issues）
图示：在推文右上角打开 HUD，查看匹配市场并点击 Trade。
如何使用
在 X 网页端浏览推文；当出现入口时，点击它即可打开面板并查看匹配市场。
遇到问题
如果入口不显示、匹配不准或 UI 异常，请到 GitHub 提交 Issue，并附上页面链接与截图。
隐私
推文内容只在本地处理，不上传服务器。详见 privacy.html。
