Discover tradable markets on X

Opinion HUD adds a small entry next to tweets on X. Open the panel to see matched markets and options, then click Trade to open the market page.
  • Button location: to the left of the “…” menu in the tweet header
  • Usage: click to open/close the panel (hover also works)
  • Panel: shows matched markets/options with a “Trade” button
View on GitHub Report an issue
Screenshot: open the HUD and click Trade.
How to use
Browse X on desktop. When the entry appears on a tweet, click it to open the panel.
Having issues?
If the entry doesn't show, matching is off, or the UI looks wrong, please file a GitHub issue with the tweet URL and a screenshot.
Privacy
Tweet content is processed locally and is never uploaded. See privacy.html.
