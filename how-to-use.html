Install from this repository (Developer mode)

This guide shows how to download the repo and load the extension into Chrome. No store install required.

  1. Download the repository: open the GitHub repo, click Code → Download ZIP, then unzip it.
  2. Open Chrome and go to chrome://extensions.
  3. Enable Developer mode (top-right).
  4. Click Load unpacked and select the extension/ folder inside the repo.
  5. Open https://x.com/ (desktop web). Refresh if needed.
  6. When a tweet matches a market, an entry appears to the left of the … menu in the tweet header. Click it to view market odds.
Tip: after pulling updates, go back to chrome://extensions and click the extension’s Reload button.
If something looks wrong, please include the tweet URL and a screenshot when filing an Issue.
